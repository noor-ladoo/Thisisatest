# Hello World Application

A simple Hello World application written in Python.

## Description

This repository contains a basic "Hello, World!" program that demonstrates the fundamentals of a simple Python application.

## Requirements

- Python 3.x

## Usage

To run the Hello World application:

```bash
python3 hello_world.py
```

Or run it directly (since it's executable):

```bash
./hello_world.py
```

## Expected Output

```
Hello, World!
```
